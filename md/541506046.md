* String.raw 是用来获取一个模板字符串的原始字符串的，比如说，占位符（例如 ${foo}）会被处理为它所代表的其他字符串，而转义字符（例如 \n）不会。

```js
var name = 'zlx'
var str = String.raw`my name is \n${name}`
str // "my name is \nzlx"

```
