 ***

一般编码中我们经常写如下判断：

```js
if(!a) {
  // do something
}

```
这个判断其实null undefined NaN '' 0都可以通过。
更精确的判断：

```js
// 判断null：if (a === null) {}
// 判断undefined：if (typeof a === 'undefined') {}
// 判断isNaN：if (isNaN(a)) {}

```
