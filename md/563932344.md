class 中super()：

```js
class A {
  constructor(params) {
    console.log('构造函数被执行了', params)
  }

}
class B extends A {
  constructor() {
   super(123)
 }
}
var b = new B()
// 构造函数被执行了 123

```
调用super()则父constructor被执行的时候会接受参数
